SMGO: Higher-dimensional test case

This example runs SMGO with the minimal required arguments, populated in the struct options. This is practically the same as the minimal example, but with dimensionality D=7.
Clear struct options to start this example.
clear options;
We supply the minimal options here, just with a 7-dimensional Deb's function #1 objective.
options.objfun = @(x)fn_deb1(x);
options.bounds = repmat([-1 1], 7, 1);
options.maxiter = 250;
Run the SMGO algorithm using the supplied arguments in options.
out = smgo(options);
Plot the resulting samples distribution throughout the search space. We only show the first two dimensions here.
plot(out.hist_x(1,:), out.hist_x(2,:), '.'); grid on, title('Sampled points (only first 2 dimensions)');
