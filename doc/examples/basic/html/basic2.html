SMGO: Using initial sampling points

This example uses SMGO with an initial set of sampling points using options.startpt. In this case, SMGO samples the supplied points set before beginning the first iteration.
Clear struct options to start this example.
clear options;
We supply the minimum required arguments for SMGO, as done in the minimal example for a 2-dimensional Styblinski-Tang objective function.
options.objfun = @(x)fn_styblinski(x);
options.bounds = repmat([-5 5], 2, 1);
options.maxiter = 250;
We desire for this example for SMGO to have 100 initial sampling points. In this regard, we supply a randomly-generated matrix of size 2 x 100, scaled to fill the search bounds using our utility function normd2real.
options.startpt = normd2real(rand(2,100), options.bounds);
We run SMGO with the options supplied.
out = smgo(options);
The results are now plotted, with the initial sampled points marked with o.
plot(out.hist_x(1,:), out.hist_x(2,:), '.'); grid on, title('Sampled points');
hold on; plot(options.startpt(1,:), options.startpt(2,:), 'o');
