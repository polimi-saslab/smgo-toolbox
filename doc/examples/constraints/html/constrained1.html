SMGO: Using a black-box constraint

This example uses SMGO with a real-valued black-box inequality constraint, supplied in the field options.confun.
Clear struct options to start this example.
clear options;
Fill struct options with the minimal options. In addition, we supply options.confun with a function handle. According to our convention, the constraint g is fulfilled when gâ‰¥0.
options.objfun = @(x)fn_styblinski(x);
options.bounds = repmat([-5 5], 2, 1);
options.maxiter = 250;
options.confun = @(x)fn_parab(x);
Run the SMGO algorithm.
out = smgo(options);
